Cubester

Time bends. Space narrows. You move.

Overview

Cubester is a mobile game I created when I was younger, a game where you would alter time to help a cube maneuver through small gaps in a variety of different levels and even different game-modes.

I planned to release it to the Google Playstore and use ads and in-game purchases to monetise the game.

My Role

To develop Cubester, I:

  • Programmed in C# using Unity
  • Created user interface
  • Created 3D models using Blender
  • Planned the development process
